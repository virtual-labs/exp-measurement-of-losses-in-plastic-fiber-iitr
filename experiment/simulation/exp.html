Measurement of Losses in Plastic Fiber

Instructions

  1. Click the 'Power' button to start the experiment.
  2. Double click the 'SineWave' button to generate input waveform.
  3. Click the 'Oscilloscope' button to display the output waveform.
  4. Use the controllers to vary the Amplitude, Frequency, and Volt/Div.
  5. Click the 'Dual' button to observe both waveforms.
  6. Channel 1 displays the input waveform, and Channel 2 displays the output waveform.
  7. Click the 'Add' button to read the input values into the table.
  8. Repeat step 7 for ten different input values.
  9. Now, calculate the loss (in dB) for each row and enter the results (up to 4 decimal places) into the table.
    Formula used to calculate loss (α):

    • α = - log 10 V1 V2 L1 + L2 * 8.686

    • where V1 and V2 are the output voltages, and L1(1m) and L2(5m) are the cable lengths.

  10. Click on 'LOSS' button to verify the calculated loss.

You can read only one value for each select option.
You cannot change the amplitude in the middle of reading values into table.
1 nepers = 8.686 dB
The correct values for loss will be updated in the table after three incorrect attempts.
